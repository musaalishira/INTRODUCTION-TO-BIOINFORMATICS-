# INTRODUCTION-TO-BIOINFORMATICS-
Introduction Bioinformatics integrates biology, computer science, and data science to analyze and interpret biological data. Modern bioinformatics research relies heavily on computational tools, programming languages, version control systems, and automated workflows to ensure reproducibility and efficient data analysis. 
Bioinformatics is a course that teaches us how to use computers and different programming languages to study biological data. In this project, we are showing how to work as a team to collect data from different scripts like Bash, R, Perl, and C++.
Our main goal is to build an automated pipeline that can take information from all these different languages and put them together in one file called results.csv. This helps in making data analysis faster and reduces errors when working with many group members.
What we did in this Project:
We used GitHub to manage our files and collaborate as a group.
Every member wrote a script in a different language to show their information.
We created a master script (run_automation.sh) to join all the data together automatically.
We used tools like Acode on our smartphones to write and test the codes.
